# FPV LoS Simulator 開発経緯要約（2026-04-17）

## 1. プロジェクトの目的
山岳地帯でのFPVドローン長距離飛行において、機体とパイロットの間の視界（LoS: Line of Sight）を確保し、信号途絶による墜落を防ぐための経路計画ツールを開発する。

## 2. 技術スタック
- **Frontend**: React (Vite) + TypeScript
- **Map Engine**: MapLibre GL JS (react-map-gl/maplibre)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Data Source**: 国土地理院タイル（標準地図、航空写真、DEM標高タイル）

## 3. 開発の主な経緯と技術的課題の解決
- **初期構成**: 2Dマップ（左）と3Dマップ（右）の左右分割レイアウトから開始。
- **地形表示のトラブルシューティング**: 
    - MapLibre標準では地理院の標高タイル形式（PNG）を直接解釈できないため、`custom`エンコーディングを使用して、RGB値から標高を算出するカスタムデコードロジック（H = (R*2^16 + G*2^8 + B) * 0.01 ...）を実装。
    - 3D表示時の404エラーを回避するため、標高タイルの有効ズームレベル（9-14）を明示的に制限。
- **UIの刷新**: ユーザーからのフィードバックに基づき、2D全画面をベースとし、3Dビューを右下の小窓（Picture-in-Picture）で表示する、操作性の高いレイアウトへ変更。
